# RiddleLoot

## Introduction
RiddleLoot is an engaging project designed to challenge users with riddles while rewarding them with loot for solving them. The aim is to enhance problem-solving skills in a fun and interactive way.

## Features
- A variety of riddles to challenge users.
- Reward system that provides loot for solved riddles.
- User-friendly interface for easy navigation.

## Installation
To get started with RiddleLoot, follow these installation instructions:
1. Clone the repository:
   ```bash
   git clone https://github.com/xXClipperXx/RiddleLoot.git
   ```
2. Navigate to the project directory:
   ```bash
   cd RiddleLoot
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Usage
To run the RiddleLoot application, use the following command:
```bash
npm start
```
